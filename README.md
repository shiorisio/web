## Usage
In web directory, run the following command.  

    python cgiserver.py

In WWW browser, read the following URL.

    http://localhost:8000/cgi-bin/test.py
    http://localhost:8000/cgi-bin/nowdate.py
    http://localhost:8000/cgi-bin/querytest.py?foo=1
